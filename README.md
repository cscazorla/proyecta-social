# Proyecta Social

Static website for **Proyecta Social** — consultancy firm specializing in non-governmental organizations.

## Stack

Plain HTML + CSS. No build step required.

## Local development

Open `index.html` in a browser, or use any static file server:

```bash
npx serve .
```

## Deployment (Netlify)

1. Push this repository to GitHub/GitLab.
2. Connect it to Netlify (or drag-and-drop the folder in Netlify Drop).
3. Configure form email notifications: **Site settings > Forms > Form notifications > Email notification**.

The contact form uses [Netlify Forms](https://docs.netlify.com/forms/setup/) — no backend needed.
